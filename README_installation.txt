Thank you for installing the conservation mode tool for Lenovo Legion 5 pro.


The installation procedure is very simple

Download both the conservation_mode.py and init.sh files and KEEP BOTH THE FILES IN THE SAME FOLDER ALWAYS.

Once done, open your terminal from the folder which has both the files

$ bash init.sh

From then onwards, to run the tool perform, 

$ conservation mode

on your terminal.
